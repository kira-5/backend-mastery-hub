    def targetIndices(self, nums: List[int], target: int) -> List[int]:
        freq = [0] * 101

Alternative Counting with Histogram

Given the small value range, we could use a histogram to count frequencies:

python
class Solution: def targetIndices(self, nums: List[int], target: int) -> List[int]: freq = [0] * 101 for num in nums: freq[num] += 1 less_count = sum(freq[:target]) target_count = freq[target] return list(range(less_count, less_count + target_count))
  • Pros: Cleaner counting using a fixed-size array.
  • Cons: Uses O(1) extra space (size 101), which is slightly less space-efficient than two variables. Time complexity remains O(n).
  • Verdict: Equivalent in time complexity, but the two-variable approach is simpler and uses less memory for this problem.

Edge Cases Handled

  • No Target: If target_count = 0 (e.g., target = 4 in [1,2,3]), the range is empty, returning [].
  • Single Target: If target_count = 1 (e.g., target = 3 in [1,2,3]), returns [less_count].
  • Multiple Targets: If target_count > 1 (e.g., target = 2 in [1,2,2]), returns consecutive indices [less_count, less_count + 1, ...].

  • Large Array: Efficient for max n = 100.
