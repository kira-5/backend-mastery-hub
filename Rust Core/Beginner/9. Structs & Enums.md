# Structs & Enums

---

## Structs

-   Structs are custom data types that let you group related data together.

**Declaration:**

```rust
struct User {
    username: String,
    email: String,
}
```

**Using a Struct:**

```rust
fn main() {
    let user1 = User {
        username: String::from("alice"),
        email: String::from("alice@example.com"),
        active: true,
    };

    println!("Username: {}", user1.username);
}
```

**Update Syntax:**

```rust
let user2 = User {
    email: String::from("bob@example.com"),
    ..user1 // reuse rest from user1
};
```

### Methods on Structs

-   Methods are functions associated with structs.
-   You can define methods for structs using `impl` blocks.

**Declaration:**

```rust
struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    fn area(&self) -> u32 {
        self.width * self.height
    }

    fn can_hold(&self, other: &Rectangle) -> bool {
        self.width > other.width && self.height > other.height
    }
}

// using methods
fn main() {
    let rect1 = Rectangle { width: 30, height: 50 };
    println!("Area: {}", rect1.area());
}
```

## Enums

-   Enums allow you to define a type by enumerating its possible variants.

**Declaration:**

```rust
enum Message {
    Quit,
    Move { x: i32, y: i32 },
    Write(String),
    ChangeColor(i32, i32, i32),
}
```

**Using an Enum:**

```rust
fn main() {
    let msg = Message::Write(String::from("hello"));
}
```

**Enums with impl Block:**

```rust
impl Message {
    fn call(&self) {
        match self {
            Message::Quit => println!("Quit!"),
            Message::Move { x, y } => println!("Move to ({}, {})", x, y),
            Message::Write(text) => println!("Text: {}", text),
            Message::ChangeColor(r, g, b) => println!("Color: {}, {}, {}", r, g, b),
        }
    }
}
```
